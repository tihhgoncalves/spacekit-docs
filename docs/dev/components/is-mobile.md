# Is Mobile

Este componente é responsável por somente exibir aquilo que será exibido em celulares.

### Exemplo de uso

```vuejs
<template>

    <section>
    
    <h1>Aparece sempre</h1>

        <is-mobile>
            <p>Acessado de um celular!</p>
        </is-mobile>

    </section>

</template>
```
